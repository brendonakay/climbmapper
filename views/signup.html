Why sign up?

Remember, this is an experimental project the is more fun to share with others than hide behind closed doors. I want other Mountain Project users to be able to explore their past and future in climbing while also helping to influence ClimbMappers future. Signing is needed to:

  1. Maintain your Mountain Project API authentication data (user email and user key).
  2. Allow the application to manage your climbing route Tick and ToDo data without overloading the Mountain Project server with data downloads each time you visit the site.
  3. Allow you to help add climbing area/crag location data which will be freely available by the public (feature coming soon).
